# Gestão de Documentos - Desafio Técnico

Sistema de gestão de documentos com upload de arquivos (PDF/JPG/PNG) e comentários.

## Tecnologias

- Python 3 + Flask
- SQLite
- HTML/CSS/JavaScript

## Funcionalidades

- Upload de documentos (PDF, JPG, PNG)
- Título e descrição
- Listagem de documentos
- Download de arquivos
- Sistema de comentários com data/hora

## Como executar localmente

```bash
pip install -r requirements.txt
python app.py
```

Acesse: http://127.0.0.1:5000

## Deploy

Configurado para deploy no Render com Gunicorn.

## Estrutura

Todos os arquivos na raiz (sem subpastas) para facilitar deploy.
